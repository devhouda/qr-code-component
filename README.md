## Overview

The challenge was to build a QR code card component and get it looking as close to the provided design as possible. The component is centered on the page and displays a QR code image with a title and a short description below it.

## Getting Started

No dependencies or build tools required.

1. Clone the repository:
   ```bash
   git clone https://github.com/devhouda/qr-code-component.git
   ```
2. Open `index.html` in your browser:
   ```bash
   open index.html
   # or double-click index.html in your file explorer
   ```
